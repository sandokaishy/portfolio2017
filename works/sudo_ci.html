Graphic Design

關於這個專案

Sudo Recruit 專注於建構工程師社群，Sudo 受到電腦終端機、文字編輯器的啟發，將此元素與 Ubuntu 字型結合，塑造出深色藍底的企業識別；因程式的起源正是編輯器與終端機的游標，因此以螢光綠色強調終端機上的游標，強調工程師的無限價值。

設計理念

Sudo 又稱 substitute user do 或是 superuser do。
Sudo 是一種程式，用於類Unix作業系統如BSD，Mac OS X，以及GNU/Linux以允許使用者透過安全的方式使用特殊的權限執行程式。

字型採用工程師熟知的 Ubuntu 字型，將字型的「d」的上半部由切角改為較為平直的形狀。

色彩設定

顏色上受到文字編輯器 Sublime 的色彩主題啟發，採用螢光綠色與深藍色。
為了因應平面輸出與螢幕上呈現的差異，在平面輸出上螢光綠色有稍作調整。

識別的使用規範

識別在使用上是自由的，深藍色的背景是可以無限延伸的，唯一需要注意的地方在於背景在限縮時（Shrink），Sudo 字樣要與背景的邊界保有最小限度的距離。

Sudo 字樣與背景上下邊界的距離為 X height，與背景右邊的邊界則是取 d 的直線筆畫 寬度。

字型使用

Sudo Recruit 主要為網路招聘服務，因此中文字型的選擇上會使用 Google 開發的思源黑體，在 Webfont 上使用 Early Access 的 Noto Sans 字型（算是思源黑體的前身）。

識別應用 - 名片

名片的印刷採用凸板印刷(Letterpress)，因此在深藍色的呈現上可以更忠於色彩設定的深藍色。
由於新創公司在預算上的限制，因此只採用單色印刷無法再選擇一色螢光綠色，這是非常可惜的地方。

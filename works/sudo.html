關於 Sudo Recruit

Sudo Recruit 以 Linux 程序為名，致力將台灣打造為下一個矽谷，成為軟體專才及網路菁英匯聚的亞洲心臟 。 Sudo Recruit 旨在提供最高規格的求職媒合服務，以專業顧問的方式，協助台灣人才站上對的舞台發光發熱。

第一階段會先建立一個求職平台，搭配現下的獵頭服務發掘更多求職洞察，蒐集足夠的求職者洞察，發展第二階段：以系統判斷後進行媒合的半自動化功能。

兩種使用者

平台上分為兩種使用者，求職者與公司人資。 公司人資與一般使用者相同，可以維護自己的個人履歷（稱之為 Profile），但與求職者不同的是他們可以創建公司資料，而公司下又可以創建職缺；因此公司人資可以維護的有公司的資料、職缺與人資個人的履歷。

顏色與字型

承襲 Sudo C.I. 的基礎規範，沿用 Sudo 的藍色與基本的思源黑體。由於螢光綠色在螢幕上會造成文字閱讀不易，所以調整原先螢光綠色的亮度，改為較為暗淡的綠色作為強調色。

Landing Page

使用者可以透過 Landing page 直接搜尋職缺，進而從職缺內容連結至詳細的公司頁面。
另外，透過上方的 navigation bar，使用者也可以直接跳到職缺列表與公司列表，輸入搜尋條件開始找尋自己偏好的公司與工作。

Job Index Page

使用者可以依照所屬類別找尋職缺，透過篩選器排列職缺。
每一筆職缺呈現的資訊除了薪資之外，更加上了技術標籤，讓求職者比較快速理解該職缺所需要的技術。

Company/ Job information

使用者可以依照所屬類別找尋職缺，透過篩選器排列職缺。
每一筆職缺呈現的資訊除了薪資之外，更加上了技術標籤，讓求職者比較快速理解該職缺所需要的技術。

Company Index

公司列表呈現公司的團隊相片，除此之外更強調公司的融資狀態、公司人數與公司產業類別。公司列表未開發較完整的搜尋功能，僅能以關鍵字搜尋公司。

Components and Mail template

由於 Sudo Recruit 為人才媒合的服務，需要寄送頻繁的的系統通知信件。
我們使用 Ruby On Rails 串 Mailgun 寄信並使用 MJML framework 撰寫 Email 模板。

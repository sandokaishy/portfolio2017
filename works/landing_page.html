Collage Landing Page

New feature introduction

Apr. 2018

UI/ UX, Frontend

About this project

Collage is a new feature of our product. It’s an editor tool let the shop owner design their online shop.
We create various kind of module: photo module, blog module, product module and so on. User can make unique visual by arranging the order of the modules.

Mission

This landing page should be an introduction for our new feature.
Because the feature is still on the beta stage. The goal is not only to make users understand how it works but to apply for this new feature.

My Role

I’m the frontend developer and designer of this project because it’s not arranged in the sprint, and frontend team doesn’t have enough time to develop this page.

Challenge

  • Limited time.
  • The feature is beta, so it’s not attractive to introduce with beta interfaces.

Solution

How to introduce the concept and attract users to apply for early access without showing the editor interface? We could change the way we communicate. Instead of the real interface, we show the page production created by Collage. We talk about what Collage can do for them, not how to use it.

Execution

After 2 days discussion of film storyboard, 2 days film making and 1 days creating mockups, it costs 8 days in total to finish the landing page.
